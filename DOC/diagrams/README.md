# Диаграммы проекта Maze Game

Эта папка содержит блок-схемы алгоритмов приложения, созданные с использованием PlantUML.

## Файлы

- `генерация_лабиринта.puml` - Блок-схема алгоритма генерации лабиринта (DFS)
- `игровой_цикл.puml` - Блок-схема основного игрового цикла
- `поиск_пути.puml` - Блок-схема алгоритма поиска пути

## Как просмотреть диаграммы

### Онлайн (самый простой способ)

1. Откройте файл `.puml` в текстовом редакторе
2. Скопируйте содержимое файла
3. Перейдите на http://www.plantuml.com/plantuml/uml/
4. Вставьте код в редактор
5. Диаграмма отобразится автоматически

### Локально с Java

1. Установите Java (если не установлена)
2. Скачайте `plantuml.jar` с https://plantuml.com/download
3. Запустите команду:
   ```bash
   java -jar plantuml.jar диаграммы/*.puml
   ```
4. Изображения будут созданы в той же папке

### Использование в IDE

**IntelliJ IDEA / Rider:**
- Установите плагин "PlantUML integration"
- Файлы `.puml` будут автоматически отображаться как диаграммы

**Visual Studio Code:**
- Установите расширение "PlantUML"
- Откройте файл `.puml` и нажмите `Alt+D` для предпросмотра

**Visual Studio:**
- Установите расширение "PlantUML Editor"

### Генерация PNG/SVG

Для генерации изображений из PlantUML файлов:

```bash
# Генерация PNG
java -jar plantuml.jar -tpng диаграммы/*.puml

# Генерация SVG
java -jar plantuml.jar -tsvg диаграммы/*.puml
```

## Описание диаграмм

### Генерация лабиринта
Показывает процесс создания лабиринта с использованием алгоритма рекурсивного обхода в глубину (DFS), включая добавление тупиков и проверку связности.

### Игровой цикл
Описывает основной цикл работы приложения: обработку событий (клавиатура, мышь), обновление состояния игры и проверку условий завершения.

### Поиск пути
Демонстрирует алгоритм поиска пути от старта до цели с использованием метода следования по стене, включая обрезку петель для оптимизации.
